## 六、解除18+内容限制方法

若进入群/频道提示「This channel can't displayed because it was used to spread pornographic content」，按以下方法解决：

安卓端：设置 → 对话设置 → 滑至底部勾选「关闭过滤」/「18+内容」，重启App即可；
iOS端：需要登录网页版，或者电报客户端，「隐私与安全」滑倒底，「关闭过滤」/「18+内容」

注意：各别国家号码解除了，也无法观看，如：印度，印尼

1.  设置 → 隐私和安全 → 两步验证：设置/修改验证密码，并绑定安全邮箱；
2.  设置 → 隐私和安全 → 隐藏手机号：减少手机号泄露风险；
